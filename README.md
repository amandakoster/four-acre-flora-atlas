# Four-Acre Flora

Interactive mapping and observation system for documenting, identifying, and managing plant life across a four-acre property.

## Technology Stack

### Frontend

- Next.js
- React
- TypeScript
- Tailwind CSS

### Hosting

- Vercel (Free)

### Database

- Supabase PostgreSQL (Free)

### Source Control

- Git
- GitHub

## Project Goals

- Track plant observations
- Organize observations by zone
- Build a searchable species catalog
- Document native and cultivated plants
- Support long-term ecological monitoring
- Create a living atlas of the property

## Planned Features

### Species Catalog

- Common Name
- Scientific Name
- Native Status
- Notes
- Photos

### Property Zones

- Zone management
- Habitat descriptions
- Filtering by zone

### Observations

- Observation date
- Species observed
- Location
- Notes
- Photos

### Future Enhancements

- GIS mapping
- Property atlas visualization
- User authentication
- Mobile support
- AI-assisted identification
